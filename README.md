# Data Structures and Algorithms in Java

This repository contains implementations of various data structures and algorithms in Java.

## Getting Started

You can clone this repository and run the tests using Maven.

```bash
git clone https://github.com/kalyan-juluri/dsa-journey.git
cd dsa-journey
mvn clean test
```
